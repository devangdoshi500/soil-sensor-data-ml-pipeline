## Overview
This repository contains a machine learning pipeline for cleaning soil sensor data, labeling soil health with a composite score, and training a Random Forest model.

## Dataset
The dataset used in this project is not included.
To run the pipeline, provide an Excel file containing soil sensor measurements and update the data path in config.py.

## Usage
To use the model, run the run_pipeline.py file after ensuring that all other relevant files are in the same folder as it.
python run_pipeline.py for usage
